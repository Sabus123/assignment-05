Question-1: What is event delegation in the DOM ?

Answer: Event delegation in the DOM is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements

Question-1: Wha is event building int the DOM?

Answer: Event building" in the DOM (Document Object Model) refers to the process of attaching event listeners to specific elements within a web page, allowing JavaScript to respond to user interactions like clicks, hovers, or key presses by triggering functions when those events occur on the designated elements within the DOM tree; essentially, it's the mechanism to create interactive behavior on a webpage by defining what actions should happen when certain events are triggered on the page.
